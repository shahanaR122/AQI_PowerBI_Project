# Air Quality Analysis Dashboard

This project analyzes Air Quality Index (AQI) trends using Power BI.

## Features

* AQI by city analysis
* Pollution trend visualization
* KPI cards
* Interactive filters
* Map visualization

## Tools Used

* Power BI
* Power Query
* Kaggle Dataset

## Dashboard Insights

* High AQI observed during winter months
* PM2.5 contributed significantly to pollution
* Major cities showed unhealthy AQI levels
